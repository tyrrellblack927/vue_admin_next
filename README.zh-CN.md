# Vue Admin Next

Vue Admin Next 是一套针对中后台应用场景的企业级 Web 应用脚手架，包含 **Vue Composition API**、**多模块设计**、**数据处理流**、**简洁的用户界面**等。你想要的都在这里，还等什么呢？

**提示**：当 Vue 3.0 正式版发布后，该项目会第一时间进行支持！
